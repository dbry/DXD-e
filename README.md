## What is this?

This is experimental code to investigate the DSD and DXD audio. Specifically, the idea of embedding the
source DSD stream inside the DXD audio stream (hence DXD-e). This would allow playback pipelines and
DAW software that only work with PCM audio to easily add DSD capability.
